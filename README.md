#JSBuild   
JSBuild command line tool

### Pre-requisites
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

### Installation
```
$ git clone git@github.com:JSBuild/jsbuild.git
$ cd jsbuild
$ yarn install
```

### Development  
*Start in debug mode*   
```
$ npm run dev 
```
*Create production build*  
```
$ npm run build
```
